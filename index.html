GIPI (Gestão Interna de Projetos Integradores)(ainda a implementar)

Meu trabalho de conclusão de curso foi um sistema baseado na matéria técnica Desenvolvimento de Projeto Integrador, é basicamente um registrador de projetos,que armazena dados básicos como título, descrição alunos líder e participantes, professores orientador e coorientador.

Allocated

Este projeto é mais simples com o intuito de gerar apenas as requisições, foi feito numa atividade que integrava as matérias Programação Orientada a Serviços, Projeto de Interface de usuário e Projeto e Desenvolvimento de Sistemas para Internet. Se trata de um sistema de reserva de salas de estudo da biblioteca.

imagens a implementar

Conta com as classes Usuário e Reserva, onde a requisição Reserva armazena o nome do usuário, o horário e a sala, contando com uma regra onde não se pode gerar uma requisição Reserva, caso já exista outra requisição com os mesmos horário e sala, evitando conflitos de horário
